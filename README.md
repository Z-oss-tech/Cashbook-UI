# SmartKhata - Cashbook UI

SmartKhata is a modern, gamified, multi-ledger financial management application designed for seamless business and personal accounting. It provides an intuitive, highly polished user interface, dynamic transaction tracking, and enterprise-grade PDF reporting.

## 🚀 Features

* **Multi-Ledger System:** Manage multiple independent cashbooks under a single account with customized icons and theme colors.
* **Premium Financial Reports:** Generate beautiful, colorful PDF reports featuring dynamic pie charts, AI-driven financial health insights, and detailed category analysis tables.
* **Smart Gamification:** iOS-style emoji feedback animations for tracking income and expenses playfully and interactively.
* **Data Security & Backup:** Robust cloud backend compatibility (Node.js/PostgreSQL) with a built-in 30-day data recovery system.
* **Seamless Authentication:** Multi-login capability, secure profile completion, and password/OTP authentication workflows.

## 🛠 Tech Stack

* **Framework:** Flutter (Dart)
* **State Management:** Provider
* **Reporting Engine:** `pdf` and `printing` packages with dynamically embedded Roboto fonts.
* **Backend Integration:** Node.js, Express, PostgreSQL
* **Typography & Layout:** Google Fonts (Inter, Manrope, Hanken Grotesk), Glassmorphism UI patterns.

## 📦 Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Z-oss-tech/Cashbook-UI.git
   ```

2. **Get dependencies**
   ```bash
   cd Cashbook-UI
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```
